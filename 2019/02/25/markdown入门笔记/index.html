什么是Markdown

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。
简单来说Markdown是可以快速排版的文档编辑器，而我们可以使用它来编写自己的技术文档和博客。
